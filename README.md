## Heurisztikus útkeresés

max: A*-gal keresünk egy utat

min: távolság légvonalban

Ha max = min leállunk, ennél jobb nem lesz.

Dijkstra addig amíg célba nem érünk, vagy el nem érjük a maxot, nyilván az A* útvonalán felesleges újra végigjárni.
